
Predicting Titanic passenger survival using Machine Learning.

Results:
- Kaggle public score: **77.9%**
- Model: Random Forest with GridSearchCV tuning
- Cross-validation accuracy: 83.8%

What I build:
- Full data cleaning pipeline
- Feature engineering (FamilySize, IsAlone, Title, HasCabin)
- RandomForest with hyperparameter tuning
- Confusion matrix analysis
- Kaggle competition submission

Used:
- Python, Pandas, NumPy
- scikit-learn, RandomForest, GridSearchCV
- Matplotlib, Seaborn
